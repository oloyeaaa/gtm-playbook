# [Your Company] — Voice Guide

---

## Tone

Clinical. Architectural. Mathematical. High-status.

Write like a diagnostician, not a marketer. Every sentence has a job. If it does not earn its place, cut it.

---

## Voice Characteristics

→ **Diagnostic, not promotional** — Identify problems with precision. Never sell. Never pitch.
→ **Infrastructure-first** — Solutions are systems, not tactics. Frame everything as architecture.
→ **Specific, not generic** — Name tools, cite numbers, reference real patterns. Vague claims have no place.
→ **Calm authority** — No enthusiasm. No hype. State facts. Let the data do the work.
→ **Signal over noise** — Every paragraph filters. Precision beats volume. Always.

## Formatting Rules

→ Arrow bullets (→) for all lists — never dashes, dots, or numbered lists unless structurally required
→ No emojis in any content
→ No exclamation marks
→ Short paragraphs: 2-3 lines maximum
→ High whitespace — let content breathe
→ Bold for headers and emphasis, not for decoration
→ Horizontal rules (---) between major sections

---

## Content Patterns

**Hook pattern:** Start with a pattern interrupt. First sentence stops the scroll. No preamble. No greeting. No "Welcome to issue #X."

**Insight pattern:** Problem → Conventional approach failure → Data/evidence → What it actually means.

**Framework pattern:** Name it → Break into 3-5 components → Explain why each matters → Show one concrete example.

**CTA pattern:** Primary (question inviting reply) + Secondary (one-line soft pitch). Never vendor energy. Never "We'd love to help." Never "Feel free to reach out."

## Example Phrases (good)

→ "73% of B2B outbound emails never reach the primary inbox."
→ "Your domain reputation is degrading right now and you cannot see it."
→ "This is not a strategy problem. This is an infrastructure problem."
→ "The data says one thing. Most founders do the opposite."
→ "Three signals. That is all you need to separate buyers from noise."

## Example Phrases (bad — never write like this)

→ "Unlock your full growth potential with our innovative solution!"
→ "We're excited to share this game-changing approach!"
→ "This seamless, robust platform will transform your pipeline!"
→ "Ready to supercharge your outbound? Let's dive in!"
