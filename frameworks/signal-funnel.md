# The Signal Funnel

[Company]'s lead routing framework. Every channel has one job. Every lead has one path.

---

## Principle

Don't mix nurture and conversion channels. LinkedIn organic builds trust and feeds the newsletter. Outreach (email + DMs) drives conversations. The free GTM audit bridges trust to revenue.

---

## The Three Lanes

### Lane 1: Authority (LinkedIn Organic)

**Purpose:** Build credibility so outreach doesn't land cold.

```
LinkedIn post (Mon/Wed/Fri)
  → Engagement (likes, comments, profile views)
  → Two exits:
      A. Ready to talk → DM conversation → Calendly → Call
      B. Not ready yet → Newsletter (link in comments/featured) → Nurture
```

**CTAs:** Diagnostic questions, "DM me if...", newsletter link in first comment or featured section. Never sell in the post.

### Lane 2: Outreach (Cold Email + LinkedIn DMs)

**Purpose:** Start direct conversations with qualified prospects.

```
Cold email (SmartLead C002) → Reply → Conversation → Calendly → Call → Close
LinkedIn DM (manual, 3-touch) → Reply → Conversation → Calendly → Call → Close
```

**CTAs:** Book a call (Calendly link). Offer the free GTM audit. Never link to newsletter — these channels are direct-to-conversation.

### Lane 3: Nurture (Newsletter)

**Purpose:** Keep warm-but-not-ready leads engaged until they're ready to talk.

```
Newsletter subscriber (from LinkedIn organic only)
  → Weekly issue with insight + CTA
  → 3+ opens/clicks = buying signal
  → Personal outreach → Free GTM audit → Calendly → Call
```

**Fed by:** LinkedIn organic posts only. Never promoted in cold emails or DMs.

---

## The Bridge: Free GTM Audit + Signal Stack Blueprint

The GTM audit is the trust-builder that converts interest into conversation. The Signal Stack Blueprint is the second deliverable — sent when the audit gets no reply. Together they form a value cascade.

```
Any lane → Interest signal → Run /so-gtm-audit → Deliver audit PDF
  → Reply? → Conversation → Calendly → Call
  → No reply (3 days) → Run /so-signal-stack-blueprint → Deliver blueprint PDF
    → Reply? → Conversation → Calendly → Call
    → No reply (5 days) → Final touch (breakup email) → Park or Drop
```

Full pipeline stages and follow-up sequences defined in `founder/playbooks/pipeline-plan.md`.

## Routing Rules

| Channel | Links to Newsletter? | Links to Calendly? | Primary CTA |
|---------|---------------------|--------------------|--------------------|
| LinkedIn post | Yes (in comments) | No (too salesy) | Engage / DM me |
| LinkedIn DM | Never | Yes (touch 3) | Book a call |
| Cold email | Never | Yes (after reply) | Reply to this email |
| Newsletter | N/A | Yes (in CTA) | Book a call / reply |
| GTM audit PDF | Never | Yes (in report) | Let's fix this together |

## Anti-Patterns (Never Do These)

1. Don't put newsletter links in cold emails or DMs — it's a detour from conversion
2. Don't put Calendly links in LinkedIn posts — too salesy, kills engagement
3. Don't email AND DM the same person in the same week — dedup rule
4. Don't send newsletter to people in active email/DM sequences — they're in conversion lane
5. Don't skip the GTM audit step — it's the trust bridge between interest and proposal
